# 🎮 Xbox Game Pass – Dashboard de Vendas de Assinaturas

Este repositório contém a base de dados e as regras analíticas utilizadas para a construção do **Dashboard de Vendas de Assinaturas do Xbox Game Pass**, com foco em faturamento, tipos de plano e produtos agregados como **EA Play Season Pass** e **Minecraft Season Pass**.

---

## 📊 Objetivo do Projeto

O objetivo deste projeto é analisar o desempenho de vendas das assinaturas do **Xbox Game Pass**, respondendo perguntas de negócio estratégicas relacionadas a:

- Faturamento total
- Tipos de planos vendidos
- Impacto da renovação automática
- Produtos adicionais agregados às assinaturas
